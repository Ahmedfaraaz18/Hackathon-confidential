# **App Name**: CampusConnect

## Core Features:

- User Authentication: Secure login/logout for teachers and students.
- Role-Based Dashboards: Customized dashboards for teachers and students.
- Attendance Tracking: Teachers can mark attendance for students.
- Attendance Reporting: Generate reports on student attendance.
- Real-time messaging: Ability to send and receive messages in real time using websockets, instead of only showing stored messages.
- Smart schedule suggestions: Based on the student's interests, suggest events, subjects, and classes the student might enjoy using an AI tool.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a sense of authority and reliability.
- Background color: Very light indigo (#E8EAF6) for a clean and professional look.
- Accent color: Electric purple (#7E57C2) to draw attention to important interactions.
- Body and headline font: 'Inter', sans-serif, for a modern, machined, objective, neutral look
- Use clear, modern icons for navigation and actions.
- Responsive layout adapting to different screen sizes.
- Subtle transitions and animations to improve user experience.
